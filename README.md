# HR Analytics Dashboard (Power BI)

This repository contains a Power BI dashboard designed to analyze HR data and provide insights into employee performance, attrition, and workforce trends.

## Project Overview

The dashboard helps organizations understand employee behavior and HR metrics using interactive visualizations. It allows users to explore workforce data and identify patterns that can support better decision-making.

## Key Features

* Employee attrition analysis
* Department-wise employee distribution
* Gender and age demographics
* Performance and salary insights
* Interactive filters and slicers for deeper analysis

## Tools Used

* Power BI for data visualization
* Data cleaning and transformation using Power Query

## Insights Provided

* Identification of departments with higher attrition rates
* Analysis of workforce demographics
* Understanding factors that may influence employee turnover

## File Included

* **HR analytics.pbix** – Power BI dashboard file containing all visuals, data models, and insights.

## Purpose

This project demonstrates the use of Power BI to transform raw HR data into meaningful insights that can support HR strategy and workforce planning.
